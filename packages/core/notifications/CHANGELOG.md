# Changelog

## 1.0.0

### Added

- Initial release of `@l8b/notifications` package
- `notifications.isEnabled()` - Check if notifications are enabled
- `notifications.getToken()` - Get notification token
- `notifications.getUrl()` - Get notification URL
- `notifications.getDetails()` - Get notification token and URL
- `notifications.sendTokenToServer()` - Helper to send token to server
