# @al8b/stdlib

Standard library modules for LootiScript. The package groups the built-in helper libraries that the language and runtime rely on for math, string, list, and JSON operations.

## Public API

- `MathLib`
- `StringLib`
- `ListLib`
- `JSONLib`

## Notes

- Consumed by `@al8b/lootiscript`.
- Keep script-level built-ins here so language and runtime integrations stay consistent.

## Scripts

```bash
bun run build
bun run test
```
