# L8B Documentation Site

This directory contains the source code for the L8B documentation site, built with [Vocs](https://vocs.dev).

## Structure

- `docs/` - Documentation content (MDX files)
- `vocs.config.ts` - Site configuration

## Development

```bash
# Install dependencies
bun install

# Start development server
bun dev
```

## Build

```bash
# Build for production
bun build
```
